# LOAN TRACKER WEB PLAN

## Goal
Build a standalone multi-user PWA web app for personal loan tracking.

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
- PWA
- Vercel deployment

## Product Direction
This is NOT connected to the mobile app.

The mobile app exists only as:
- UX inspiration
- design reference
- terminology reference

## Main Features
- authentication
- dashboard
- loan management
- payment history
- archive
- backup/export
- responsive mobile-first UI

## PWA Direction
The app should:
- install nicely on iPhone
- feel app-like
- support standalone mode
- prioritize touch UX

## Development Strategy

### Phase 1
- foundation
- layout
- auth shell
- navigation
- responsive UI structure

### Phase 2
- Supabase auth
- loan CRUD
- dashboard metrics

### Phase 3
- payment flow
- archive
- reschedule
- export/backup

### Phase 4
- analytics
- charts
- advanced insights
